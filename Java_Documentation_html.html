Introduction

Java is a class-based, object-oriented programming language designed to be platform-independent. It follows the principle of "Write Once, Run Anywhere" (WORA), meaning compiled Java code can run on all platforms that support Java without recompilation. Java is a strongly typed language that provides automatic memory management through garbage collection.

Java contains a comprehensive standard library of classes and interfaces, known as the Java API, which includes collections, input/output operations, networking capabilities, graphical user interface components, and much more.Core Java can be extended for various purposes:

  • Java SE (Standard Edition) provides the core functionality for desktop and simple server applications
  • Java EE (Enterprise Edition) adds capabilities for large-scale, multi-tiered, scalable, reliable, and secure network applications
  • Java ME (Micro Edition) provides an API for mobile device programming

JavaScript and Java

Despite the similar names, Java and JavaScript are very different languages:

  • Java is compiled to bytecode and runs on a Java Virtual Machine (JVM)
  • Java is strongly typed, requiring explicit declaration of variable types
  • Java is purely object-oriented; everything must be within a class
  • Java has static type checking at compile time
  • Java supports true multithreading
Hello world

Here's a simple "Hello World" program in Java:

public class HelloWorld {
public static void main(String[] args) { System.out.println("Hello, World!");
}
}

Every Java program must have a class and a main method to run.

Variables

In Java, all variables must be declared with a specific type before they can be used. Variable names (identifiers) must follow these rules:

  • Must begin with a letter, dollar sign ($), or underscore (_)
  • Subsequent characters can be letters, digits, dollar signs, or underscores
  • Case sensitive
  • Cannot use Java keywords
Declaring variables

Variables in Java are declared with their type:

int number = 42;
String text = "Hello";
double decimal = 3.14;
boolean flag = true;

Java supports three kinds of variable declarations:

  • Instance Variables (Non-Static Fields)
  • Class Variables (Static Fields)
  • Local Variables
Variable scope

Java has several levels of variable scope:

  • Class level scope (fields)
  • Method level scope (local variables)
  • Block level scope (variables in loops or conditional statements)
  • Instance level scope (non-static fields)
Data Types

Java has two categories of data types:

  1. Primitive Data Types:
    • byte (8 bits)
    • short (16 bits)
    • int (32 bits)
    • long (64 bits)
    • float (32 bits)
    • double (64 bits)
    • boolean (true/false)
    • char (16 bits)
  2. Reference Types:
    • Classes
    • Interfaces
    • Arrays
    • Enums
Control Flow Statements

if...else statement

if (condition) {
// code block executed if condition is true
} else if (anotherCondition) {
// code block executed if anotherCondition is true
} else {
// code block executed if all conditions are false
}

while statement

while (condition) {
// code block to be executed while condition is true
}

for statement

for (initialization; condition; increment/decrement) {
// code block to be executed
}
Methods (Functions)

In Java, methods are always part of a class. A method declaration includes:

  • Access modifier (public, private, protected)
  • Return type (or void)
  • Method name
  • Parameters (if any)
  • Method body

Example:

public int multiply(int a, int b) {
return a * b;
}

Key characteristics of Java methods:

  • Must be declared within a class
  • Specify a return type (or void)
  • Can have access modifiers
  • Can throw exceptions
  • Can be overloaded (same name, different parameters)
Reference

Some notable differences from other languages:

  • Java is statically typed
  • All code must be in classes
  • Memory management is automatic
  • Platform independence through bytecode
  • Rich standard library (Java API)
  • Strong support for multithreading
  • Built-in security features
